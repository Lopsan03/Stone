# 🚀 Quick Reference - 5 Minute Demo

## The Demo in One Page

### 1️⃣ CREATE CONTRACT (30 seconds)
- Tab: **Contracts**
- Click: **"Quick Demo"** button
- Click: **"Create Contract"** button
- ✓ Contract appears below

### 2️⃣ LOCK FUNDS (1 minute)
- Click: **"Deposit Escrow"** button
- Wait: 2 second animation
- ✓ Status: **"Active Engagement"** (purple badge)
- ✓ Shows: **8,000 MON Escrowed**
- ✓ Shows: **100% Locked** (amber bar)

### 3️⃣ PHASE 1 COMPLETION (1 minute)
- Switch: **DemoSwitcher** → Freelancer mode
- Click: **"Mark Done"** on Phase 1
- ✓ Status: **"Awaiting Review"** (blue badge)
- Switch: Back to Client mode
- Click: **"Approve & Release"**
- ✓ Status: **"Paid"** ✅
- ✓ Progress updates: Locked ↓ Released ↑

### 4️⃣ EXPLAIN SECURITY (1:30 minutes)
- Scroll: **"Escrow Security Guarantee"** panel
- Read: The 4 security features
- Point: **"Nadie puede escapar"** section
- Point: **"Nadie puede hacer trampa"** section

### 5️⃣ OPTIONAL - PHASE 2 (1 minute)
- Switch: Freelancer mode
- Click: **"Mark Done"** on Phase 2
- Switch: Client mode
- Click: **"Approve & Release"**
- ✓ Shows progress: 50% locked, 50% released

---

## Key Visuals to Point At

### Lock Icons 🔒
Indicates funds are locked in escrow, not accessible

### Amber Progress Bar
= Funds still locked (can't be withdrawn)

### Emerald Progress Bar
= Funds released (paid out successfully)

### Badges
- 🟡 "Awaiting Escrow Funding" = Need to deposit
- 🟣 "Active Engagement" = Funds locked, work ongoing
- 🔵 "Awaiting Review" = Freelancer done, waiting approval
- 🟢 "Paid" = Work approved, money released
- 🔴 "Dispute" = Under arbitration

## What NOT to Do

❌ Don't try to create a contract manually - use "Quick Demo"
❌ Don't use Freelancer mode for contract creation
❌ Don't approve phases without switching to client mode first
❌ Don't skip the security panel explanation
❌ Don't worry about disputes - just show the panel

---

## What TO Do

✅ Use "Quick Demo" button (instant demo data)
✅ Show both locked and released states
✅ Switch between Client/Freelancer modes
✅ Point at visual indicators (locks, bars, badges)
✅ Read the security guarantees
✅ Emphasize: "Código, no confianza" (Code, not trust)

## Troubleshooting Quick Fixes

**Form won't submit?**
- Click "Quick Demo" to auto-fill

**Contract not appearing?**
- Make sure you're in Client mode
- Check Contracts tab
- Look in "Active Contracts" section

**Can't click "Mark Done"?**
- Must switch to Freelancer mode first

**Progress bar not updating?**
- Refresh page if needed
- Make sure you approved the milestone
